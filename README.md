# 📊 Student Performance Analysis

## 📌 Project Overview
This project analyzes a dataset of student exam performance to understand how different factors such as gender, parental education, and test preparation affect student scores in math, reading, and writing.

The project is built using Python and focuses on Exploratory Data Analysis (EDA) and data visualization.

---

## 🎯 Objectives
- Analyze student performance in exams
- Compare scores across different subjects
- Study the effect of test preparation course
- Understand impact of parental education on performance
- Visualize data patterns using graphs

---

## 📂 Dataset Information
The dataset contains 1000 student records with the following features:

- Gender  
- Race/Ethnicity  
- Parental level of education  
- Lunch type  
- Test preparation course  
- Math score  
- Reading score  
- Writing score  

---

## 🛠️ Tools & Libraries Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 🔍 Analysis Performed
- Data loading and inspection  
- Checking missing values  
- Statistical summary using `.describe()`  
- Group-based analysis:
  - Gender vs Scores  
  - Parental education vs Math score  
  - Test preparation vs performance  
- Data visualization using histograms and bar plots  

---

## 📈 Key Insights
- Students who completed test preparation performed better overall  
- Reading and writing scores are generally higher than math scores  
- Parental education level has some influence on math performance  
- Score distributions are fairly balanced  

---

## 📊 Visualizations
The project includes:
